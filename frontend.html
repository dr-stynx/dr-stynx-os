ℹ️ About Dr. Stynx OS

Dr. Stynx OS is an AI assistant platform providing self-awareness, GPU monitoring, task orchestration, and semantic memory. It communicates via the Model Context Protocol (MCP) over SSE.

📡 MCP Endpoints:
• SSE Stream: /sse
• Messages: /messages/
• Dashboard: /
✨ Features:
• Real-time GPU monitoring
• Persistent state management
• Task queue orchestration
• Self-awareness heartbeat loops
• Semantic memory (Mem0) 
Available MCP Tools:
  check_gpu      — GPU stats via nvidia-smi
  heartbeat      — Self-awareness pulse
  get_state      — Current OS state
  add_task       — Add to task queue
  clear_tasks    — Clear task queue
  memory_store   — Store a memory
  memory_search  — Semantic memory search
  memory_list    — List all memories
  memory_delete  — Delete a memory
  memory_clear   — Clear all memories
  memory_stats   — Memory statistics
