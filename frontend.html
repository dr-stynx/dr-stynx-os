ℹ️ About Dr. Stynx OS

Dr. Stynx OS is an AI assistant platform providing self-awareness, GPU monitoring, and task orchestration capabilities. It integrates with Metatron for distributed data-oriented computing.

📡 API Endpoints:
• JSON-RPC HTTP: http://localhost:8080/
• WebSocket Stream: ws://localhost:8081/
✨ Features:
• Real-time GPU monitoring (NVIDIA RTX 3090)
• Persistent state management
• Task queue orchestration
• Self-awareness heartbeat loops 
Available JSON-RPC Methods:
  POST / - health, gpu.status, state.get, task.add, task.clear, heartbeat, task.list

Example Request:
{
  "jsonrpc": "2.0",
  "method": "gpu.status",
  "params": {},
  "id": null
}
